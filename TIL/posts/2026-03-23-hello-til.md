## Why This Exists

I started this TIL blog as part of my job search as a Senior Web
Application Developer. The idea is simple — write down one thing
I learn every day, no matter how small.

## What I Learned Today

Setting up a Vite + React + TypeScript PWA with `vite-plugin-pwa`
is surprisingly clean in 2026. The scaffold from `npm create
@vite-pwa/pwa@latest` handles most of the heavy lifting.

Key things I wired up today:

- `gray-matter` for parsing Markdown frontmatter
- `marked` for rendering Markdown to HTML
- `fuse.js` for client-side search
- `date-fns` for date formatting and reading time
- Dark mode via CSS variables and `data-theme` on the root element

## The Stack

// vite.config.ts — the key PWA config
VitePWA({
