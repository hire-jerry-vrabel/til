<CubsWTracker />

## How It Works

The tracker pulls live Cubs game data from the MLB Stats API —
a free, no-auth-required API that MLB maintains publicly.
It shows the current record, the last game score, and most
importantly — whether to fly the W flag or the L flag.

The W flag is an actual Chicago Cubs tradition. After every
win at Wrigley Field they raise a white W flag from the
scoreboard. After a loss it's a blue L. Every Cubs fan in
Chicago knows exactly what it means when they see it
from the rooftops across Waveland Avenue.

## The Stack

Built as a React MDX component — drop `<CubsWTracker />` in
any post and it fetches live data and renders. CSS animations
for the waving flag, confetti on a win, streaks, score bugs.

No backend. No API key. Just the MLB Stats API and React.
