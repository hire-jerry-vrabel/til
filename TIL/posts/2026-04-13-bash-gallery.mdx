Every dev blog needs a dog. This is Bash.

I built a swipe-friendly gallery component using **Swiper.js** for the main carousel and **yet-another-react-lightbox** for the fullscreen experience. Swipe left/right on mobile, arrow keys on desktop, tap any photo to enlarge with pinch-to-zoom.

## How It Works

The `<Gallery />` component takes an `images` array and renders a Swiper carousel with a thumbnail strip. Clicking any slide opens YARL's lightbox with zoom, slideshow, and counter plugins.

The full gallery lives at [/bash](/til/bash) — it opens straight into the lightbox so you get the full immersive experience immediately.

Managing 337 photos is handled by a Python script — `npm run bash:sync` renames everything sequentially and regenerates the page. Drop new photos in, run the script, push.
