# Mermaid Diagrams in MDX

Today I added inline **Mermaid diagram** support to the blog. Drop a `<Mermaid>` component anywhere in a post and it renders a diagram from a simple text description.

## Flowchart

<Mermaid chart={`flowchart TD
  A[Write MDX Post] --> B{Has Diagram?}
  B -- Yes --> C[Add Mermaid Component]
  B -- No --> D[Just Write Prose]
  C --> E[Renders as SVG]
  D --> E
  E --> F[Push to GitHub Pages]`} caption="Post publishing flow" />

## How It Works

Just one package — `mermaid` — lazy-loaded on first render so it doesn't impact page load. The component watches `ThemeContext` and re-renders with `dark` or `default` theme automatically.

\`\`\`jsx
<Mermaid chart={`graph TD; A-->B; B-->C`} caption="Optional caption" />
\`\`\`
