# Post Images

Store images for blog posts in this directory.

## Usage in posts

Reference local images using the /til/images/ path:
```markdown
![alt text](/til/images/your-image.png)
```

External images work with full URLs:
```markdown
![alt text](https://example.com/image.png)
```

## Naming convention

Use kebab-case and include the post date prefix:
```
2026-03-24-veeva-diagram.png
2026-03-24-system-design-chat.png
```
